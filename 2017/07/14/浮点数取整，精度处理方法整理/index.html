Jul 14, 2017 | JavaScript | 阅读

整理一下浮点数数据的取整、保留指定小数位数的相关操作。

取整

四舍五入

  • Math.round(x)

向上取整

  • Math.ceil(x)

  • parseInt() 适用于负数

向下取整

  • Math.floor(x)

  • parseInt(x) 适用于正数

小数保留指定位数

四舍五入

  • NumberObject.toFixed(num)
    调用 NumberObject.toString() 方法，返回 NumberObject 的字符串表示。舍入规则是四舍五入，参数 num 指定保留的小数位数。
