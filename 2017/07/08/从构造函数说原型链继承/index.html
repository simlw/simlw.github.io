新对象 vehicle 的原型（prototype）被设置为 Vehicle.prototype，对象的原型链就是在这个时候形成的。 vehicle 对象会继承 Vehicle.prototype 对象的所有属性。
JavaScript 中的继承分为两种：

  • 继承 Vehicle.prototype 对象的属性
    这种继承是一种 delegate，它的实例对象会共享其属性，但并不会克隆一份到新的对象中。当改变 Vehicle.prototype 的属性的时候，实例对象 vehicle 的属性值也会被改变。

  • 继承构造函数内部定义的属性或方法
    这种继承会把构造函数的属性克隆一份给实例对象，改变构造函数的属性，并不会影响实例对象的属性。
